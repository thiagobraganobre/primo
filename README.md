# Números Primos - Nova Abordagem Didática

    Através do conceitos de blocos podemos fazer uma valiadação didática se um número é composto ou primo, para isso organizamos o número inicial em pares e fazemos uma organização até que localizemos seus fatores, ou que após a exaustiva procura voltemos a primeira coluna constatando que o produto por não ter fatores é um primo.

    Ainda não foi trabalhado otimização do algoritmo, e por trazer uma abordagem didática o estudo ainda continua em desenvolvimento.
